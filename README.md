# Next-Gen Robotics: Real-Time Command and Control via 5G Edge AI

## Overview
This project is developed as part of an ITP (Integrated Team Project) for Computer Engineering (CEG) students at the Singapore Institute of Technology (SIT).

The goal is to enable real-time command and control of a robotic system using 5G and Edge AI concepts.

## Hardware
- Yahboom Car Robot, Laptop 

## Implementation
- Devlop multiple Python scripts for real-time command and control
- Low-latency communication with the robot

## Key Features
- Real-time control
- Low latency
- Practical robotics implementation

## Application
- Smart robotics
- Remote control systems
- Automation
